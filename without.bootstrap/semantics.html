All HTML Tags

<abbr>

Defines an abbreviation

<b>

Defines bold text

<bdi>

Isolates a part of text that might be formatted in a different direction from other text outside it

<bdo>

Overrides the current text direction

<br>

Defines a single line break

<cite>

Defines the title of a work

<code>

Defines a piece of computer code

<data>

links a given content with a machine-readable translation

<dfn>

Represents the defining instance of a term

<em>

Defines emphasized text

<i>

Defines a part of text in an alternate voice or mood

<kbd>

Defines keyboard input

<mark>

Defines marked/highlighted text

<q>

Defines a short quotation

<rp>

Defines what to show in browsers that do not support ruby annotations

<rt>

Defines an explanation/pronunciation of characters (for East Asian typography)

<rtc>

Embraces semantic annotations of characters presented in a ruby of rb elements used inside of ruby element.

<ruby>

Defines a ruby annotation

<s>

Defines text that is no longer correct

<samp>

Defines sample output from a computer program

<small>

Defines small text

<span>

Specifies a section in a document

<strong>

Defines strong text

<sub>

Specifies subscripted text

<sup>

Defines superscripted text

<time>

Defines a date/time

<u>

Specifies text with a non-textual annotation.

<var>

Defines a variable

<wbr>

Specifies a line break opportunity for very long words and strings of text with no spaces.
